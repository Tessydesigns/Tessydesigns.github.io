About me

I’m a front-end developer who builds clean, responsive web interfaces using HTML, CSS, and JavaScript. I enjoy creating interactive features, working with APIs, and turning simple ideas into usable applications like calculators and weather apps. Next, I want to strengthen my JavaScript skills, build a strong portfolio, and move into a junior front-end role where I can work on real-world projects and continue learning.

My approach

My approach as a front-end developer is to start by understanding the problem and the user’s needs before writing any code. I break features into small, manageable parts and build them step by step using semantic HTML, responsive CSS, and clear JavaScript. I test frequently in the browser, focus on accessibility and usability, and refine the interface based on feedback. I aim to write clean, maintainable code and continuously improve through practice and learning.
